# About this image

Docker image for [DocFX](http://dotnet.github.io/docfx/).

# Spec

| Product | Version |
|--------|---------|
| DocFX  | 2.19  |

# How to use

- for build docfx-project

```sh
cd <docfx_project_dir>
docfx build
```

- for serve

```
cd <docfx_project_dir>
docfx docfx.json --serve
```
